# General Upgrades

====== NOTES ======

These upgrades are not tied to specific facilities. They provide global bonuses, meta-progression rewards, and special effects based on overall gameplay.

**Categories:**
- Problem Mastery: Bonuses for solving specific problem types
- Bonus Events: Improve bonus problem frequency and rewards
- Hack Defense: Make hack events easier or more rewarding
- Meta Progression: Rewards for time played, problems solved, etc.
- Easter Eggs: Hidden unlocks and references
