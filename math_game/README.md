# Computation Factory

Math-based incremental game. Solve problems → build computing empire.

**NOTE: Prestige and achievements NOT implemented yet.**

---

## Overview

| Stage | Aesthetic | Problems |
|-------|-----------|----------|
| **1** | Mental (human) | Arithmetic |
| **2** | Analog (mechanical) | Probability |
| **3** | Digital (electronics→AI) | Linear Algebra |
| **4** | ASI (cosmic) | Advanced Mixed |

---

## Scaling (Cookie Clicker Inspired)

**Cost**: `baseCost × 1.15^owned`

**Ownership Tiers**:
- 1: Base | 10: 2× | 25: 4× | 50: 8× | 100: 16× | 200: 64×

**Upgrades per Building**: 4-5 (at tiers 2/3/4/5 + synergy)

**Golden Problems**: Random bonuses (2× production, 10× solve, time warp, jackpot)
