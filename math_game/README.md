# Computation Factory

A math-based incremental game where you solve problems to build a computing empire.

## Facilities
Purchasable buildings that generate computations over time.

- **Tally Sticks** (Stage 1): Primitive counting tools. (+0.05 comps/sec)
- **Pencil & Paper** (Stage 1): Basic notation for performing calculations. (+0.3 comps/sec)
- **Abacus** (Stage 2): Ancient mechanical calculator. (+1.5 comps/sec)
- **Slide Rule** (Stage 3): Analog computer for multiplication and division. (+6 comps/sec)
- **Monkey Typist** (Stage 4): Infinite patience producing random digits. (+25 comps/sec)
- **Spreadsheet** (Stage 5): Grids of cells for batch processing. (+100 comps/sec)

## Upgrades
enhancements that improve efficiency, solve value, and other improvements.

- **Eraser**: Undo mistakes. (+10% solve value)
- **Ballpoint Pen**: Smoother writing flow. (+15% solve value)
- **Mental Training**: Cognitive exercises. (+25% solve value)
- **Flashcards**: Memorization aids. (+20% facility efficiency)
- **Caffeine**: Chemical alertness. (+30% passive speed)
- **Focus Music**: Lo-fi beats to study/compute to. (+35% combo bonus)
- **Ergonomics**: Better posture, better results. (+40% facility efficiency)

## Progression Stages

The game evolves through four main eras of computation.

### 1. Mental Stage
The foundation of all computing. Solving problems using biological neural networks (your brain).

**Substages:**
(Unlocked stages are added to the pool of possible problems. All previously unlocked problem types have an equal chance of appearing.)

1.  **Basic Addition**: 1-digit + 1-digit.
2.  **Basic Subtraction**: 1-digit - 1-digit (positive results).
3.  **Basic Multiplication**: 1-digit × 1-digit.
4.  **Double Addition**: 2-digit + 2-digit.
5.  **Triple Term Mixed (Add/Sub)**: 1-digit ± 1-digit ± 1-digit.
6.  **Scalar Multiplication**: 1-digit × 2-digit.
7.  **Simple Division**: 2-digit ÷ 1-digit (integer result).
8.  **Triple Term Mixed (Mul/Add/Sub)**: 1-digit ops 1-digit ops 1-digit.
9.  **Triple Addition**: 3-digit + 3-digit.
10. **Large Scalar Multiplication**: 1-digit × 3-digit.
11. **Triple Subtraction**: 3-digit - 3-digit.
12. **Large Division**: 3-digit ÷ 1-digit.
13. **Complex Mixed**: 1-digit op 2-digit op 1-digit (Add/Sub/Mul/Div).
14. **Double Multiplication**: 2-digit × 2-digit.
15. **Advanced Triple Mixed**: 2-digit op 2-digit op 2-digit (Max 1 multiplication).
16. **Quad Addition**: 4-digit + 4-digit.
